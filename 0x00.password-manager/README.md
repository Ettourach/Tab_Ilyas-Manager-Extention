# Password Manager Chrome Extension

This is a Chrome extension that functions as a **password manager**. It allows you to securely store, generate, and manage passwords for various websites. The extension also integrates with a backend server that uses a SQL database to securely save your passwords.

## Features
- **Password Generation**: Generate strong and secure passwords.
- **Password Storage**: Save passwords to a SQL database via a backend server.
- **Autofill Support**: Automatically fill login forms with saved passwords (this feature can be added in future updates).
- **Secure Storage**: Passwords are stored in a backend server, ensuring that sensitive data is securely managed.

## Technologies Used
- **Frontend**: Chrome Extension (HTML, CSS, JavaScript)
- **Backend**: Node.js with Express
